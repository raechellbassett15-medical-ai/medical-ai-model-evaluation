# medical-ai-model-evaluation
Evaluation of AI model outputs for medical coding accuracy.

Focus areas:

• Coding accuracy of ICD-10-CM, ICD-10-PCS and CPT  
• DRG classification logic  
• Clinical reasoning errors  
• Hallucinated coding guidance

Each case includes:

1. model prompt 
2. model output
3. error analysis
4.  correct coding reasoning

Purpose: 

To demonstrate SME-level evaluation used in training and benchmarking healthcare AI systems.
